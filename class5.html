Activity

  1. Add the viewport meta tag to your index.html file
  2. Add a table to your index.html file
    • The table should have 4 rows and 2 columns
    • The first row should span 2 columns and say "News Sites"
    • Fill in the remaining 6 cells with links to news sites
    • Add the target attribute to your anchor tags so that the links open in a new window/tab
  3. When you have tested all the links and your table looks good, commit your changes and push them to GitHub
